Bootstrap is the popular HTML, CSS and JavaScript framework for developing a responsive and mobile friendly website.

Why we use Boostrap

  • Increase Development Speed
  • Responsiveness
  • Consistency
  • Customizable
  • Support

How to use

  • It is absolutely free to download and use its a open source
  • It is a front-end framework used for easier and faster web development
  • CSS plug-ins and It can also use JavaScript plug-ins
  • It facilitates complete responsive designs
  • IT contains a lot of custom jQuery plugins
  • These components are customizable and you can customize Bootstrap's components
  • Bootstrap current version is v3.3.7
  • Bootstrap next or beta version is v4.0
